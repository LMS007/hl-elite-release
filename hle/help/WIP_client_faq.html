Q: I can not join a team, I clicked on a team but I am still in spectator mode.

A: The issue might be with your configuration file. Check your config file for "setinfo" commands and delete them. HLE uses more buffer than Valve or AG due to additional features.

New Client Commands

Command Description
help Opens help menu and lists all commands and variables.
menu Opens the VGUI menu (bind this in the launch menu).
drop Reveals a drop menu with drop commands. Requires mp_tossitem set to 1.
yes Vote yes.
no Vote no.
